# Dossier Images

## Fichiers à ajouter :

1. **photo-profil.jpg** : Votre photo principale (recommandé : 800x1000px ou ratio similaire)
   - Placez ce fichier directement dans ce dossier
   - Nom exact : `photo-profil.jpg`

2. **projets/** : Images de vos projets
   - Ajoutez les images de vos projets dans ce sous-dossier
   - Formats acceptés : .jpg, .png, .webp
   - Recommandé : 1200x800px pour un bon affichage

## Note
Si une image n'est pas trouvée, le système utilisera automatiquement un placeholder.
